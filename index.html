Philosophie

 

Gute Webseiten müssen nicht nur gut aussehen.

Sie müssen auch auf allen Geräten für den User ein perfektes Erlebnis bieten.

Wir sorgen mit Ihnen gemeinsam dafür, dass diese Anforderung erfüllt wird.

Und bieten dabei ein Komplett Lösung von der Planung bis zu laufenden Betreuung.

Der Weg zur Website

 

Planung

Ein gute Website beginnt bei der Planung. Ausgehend von den Inhalten und der Zielgruppenanalyse erarbeiten wir in einem gemeinsamen Workshop die künftige Website.

Design

Steht das Ziel fest, kann mit dem Design begonnen werden. Beginnend bemit dem erstellen der Struktur mit Wireframes folgen erste Mockups und Prototypen. Erste User Test können in dieser Phase bereits zeigen, ob der eingeschlagene Weg erfolgreich ist.

Umsetzung

In diesem Schritt wird die Website mit HTML, CSS und Java Script zu einer fertigen Website ausgebaut und auf Wunsch in ein CMS eingebunden. Anschliessend kann mit dem befüllen der entgültigen Inhalte begonnen werden. Wir bereiten dabei die Inhalte vor allem Fotos und Videos so auf, damit diese für das Web optimal geeignet sind.

Betreuung

Eine Website benötigt auch eine laufende Betreuung. Wir sorgen für die optimale Erreichbarkeit und die notwendigen Updates der verwendeten Software. Auf Wunsch pflegen wir auch neue Inhalte ein. Vermarktung und laufende Suchmaschinenoptimierung sind nur einige der laufenden Angebote.
